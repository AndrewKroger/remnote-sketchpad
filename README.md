# RemNote Sketchpad Plugin

A sketchpad widget for RemNote that lets you draw sketches during flashcard review and save them directly to your cards. **Optimized for iPad and mobile devices.**

## Features

- **Drawing Tools**: Pen and eraser with pressure sensitivity (Apple Pencil supported)
- **Color Palette**: 7 colors to choose from
- **Grid Background**: Faint grid lines for alignment
- **Auto-Clear**: Canvas automatically clears when you move to the next flashcard
- **Hint Mode**: Show the back-of-card image at 10% opacity as a tracing guide
- **Mobile Optimized**: On iPad/iOS/Android, opens as a proper pane that displaces content instead of floating
- **Two Modes**:
  - **Floating Sidebar**: Toggle sketchpad with toolbar button during flashcard review
  - **Tag Mode**: Sketchpad appears under cards tagged with "sketch"

## Installation

### From Plugin Marketplace
1. Open RemNote → Settings → Plugins
2. Search for "Sketchpad"
3. Click Install


## Usage

### Floating Sidebar Mode (Default)
1. Start a flashcard review session
2. Click the ✏️ button in the queue toolbar (top right)
3. Draw your sketch in the sidebar
4. Click "Save" to save as a child of the current card
5. The canvas auto-clears when you rate the card

### Tag Mode
1. Go to Settings → Plugins → Sketchpad → Change mode to "Tag Mode"
2. Tag any flashcard with a Rem containing "sketch" in its name
3. The sketchpad will appear under those cards during review

### Hint Feature
- Click the "Hint" button to show the back-of-card image at 10% opacity
- Great for tracing diagrams from memory
